# PWA Installability Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make Brasileirão Lendário installable as a PWA that works fully offline (except multiplayer, which already requires network).

**Architecture:** `vite-plugin-pwa` generates the manifest and service worker at build time (Workbox `generateSW` strategy, precaching the built app shell). A small Node script rasterizes a hand-authored SVG icon (sharp) into the PNG sizes the manifest needs. Google Fonts CDN `<link>` tags are replaced with `@fontsource` npm packages so fonts are bundled and precached like any other asset. `App.jsx` gets a small amount of new state to surface an "Instalar app" button using the native `beforeinstallprompt` event, plus a one-time text hint for iOS Safari (which has no such event).

**Tech Stack:** `vite-plugin-pwa` (manifest + service worker), `sharp` (icon rasterization, dev-only), `@fontsource/fraunces` / `@fontsource/source-sans-3` / `@fontsource/space-mono` (self-hosted fonts).

## Global Constraints

- This is part of a larger 3-plan initiative (Copa Lendária mode, Achievements/Carteirinha, and this PWA plan). This plan is fully independent of the other two — it does not depend on Copa Lendária or Achievements, and neither of them depends on it.
- Theme color across manifest, icons, and meta tags: `#0B1A12` (existing app background). Accent/gold: `#d4a23c`.
- Do not touch the existing simulation engine, draft logic, or multiplayer code — this plan only touches build config, `index.html`, `src/main.jsx`, and the `Intro` component / top of `App()` in `src/App.jsx`.
- No test framework exists in the repo yet. This plan uses manual browser verification (documented per-task and in the final task) rather than automated tests, since there is no pure logic here to unit test — it's all build config and a native browser API.

- [ ] **Step 3: Write the icon generator script**

Create `scripts/generate-pwa-icons.mjs`:

```js
import sharp from 'sharp';
import { mkdirSync } from 'node:fs';
import { fileURLToPath } from 'node:url';
import path from 'node:path';

const __dirname = path.dirname(fileURLToPath(import.meta.url));
const srcSvg = path.join(__dirname, 'icon-source.svg');
const outDir = path.join(__dirname, '..', 'public', 'icons');

mkdirSync(outDir, { recursive: true });

const targets = [
  { file: 'icon-192.png', size: 192 },
  { file: 'icon-512.png', size: 512 },
  { file: 'icon-maskable-512.png', size: 512 },
  { file: 'apple-touch-icon.png', size: 180 },
];

for (const { file, size } of targets) {
  await sharp(srcSvg)
    .resize(size, size)
    .png()
    .toFile(path.join(outDir, file));
  console.log(`wrote public/icons/${file} (${size}x${size})`);
}
```

- [ ] **Step 4: Add the npm script**

In `package.json`, add to `"scripts"`:


- [ ] **Step 1: Add install-prompt state to `App()`**

In `src/App.jsx`, inside `export default function App() {`, right after the existing `const [myTeamLogo, setMyTeamLogo] = useState(null);` / `const [cropSrc, setCropSrc] = useState(null);` block (around line 1950-1951), add:

```js
  // PWA — prompt de instalação (Android/desktop) e hint pro iOS
  const [installPromptEvent, setInstallPromptEvent] = useState(null);
  const [iosHintDismissed, setIosHintDismissed] = useState(
    () => localStorage.getItem('bl_ios_install_hint_dismissed') === '1'
  );

  useEffect(() => {
    const handler = (e) => {
      e.preventDefault();
      setInstallPromptEvent(e);
    };
    window.addEventListener('beforeinstallprompt', handler);
    return () => window.removeEventListener('beforeinstallprompt', handler);
  }, []);

  const isIosDevice = () => /iphone|ipad|ipod/i.test(window.navigator.userAgent);
  const isStandalone = () =>
    window.matchMedia?.('(display-mode: standalone)').matches || window.navigator.standalone === true;

  const handleInstallClick = async () => {
    if (!installPromptEvent) return;
    installPromptEvent.prompt();
    await installPromptEvent.userChoice;
    setInstallPromptEvent(null);
  };

  const dismissIosHint = () => {
    localStorage.setItem('bl_ios_install_hint_dismissed', '1');
    setIosHintDismissed(true);
  };
```
